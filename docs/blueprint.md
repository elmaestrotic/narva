## Style Guidelines:

- Primary color: Deep blue (#3468C0) to reflect the academic environment of EAFIT.
- Background color: Very light gray (#F2F2F2) to provide a clean and modern feel.
- Accent color: Soft blue (#7BB0D8), for interactive elements like buttons and links, adding a subtle touch of engagement.
- Headline font: 'Playfair', a modern sans-serif similar to Didot, for titles.
- Body font: 'PT Sans', a humanist sans-serif font for body text and descriptions.
- Use modern, clean icons to represent course features.
- Clean and structured layout with ample white space to make the content easily readable.
